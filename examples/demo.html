terminal-slide

Present from your terminal. No fuss.

Press → or Space to continue

Features

Markdown Slides

Write slides in Markdown, render beautifully in the terminal with syntax highlighting.

HTML Presentations

Serve animated HTML slides via a local server, opened automatically in your browser.

Smooth Transitions

Slide-in and dissolve effects powered by tachyonfx for polished navigation.

Keyboard Driven

Navigate with arrow keys, vim bindings, or Space. Quit anytime with q or Esc.

Quick Start

# Install terminal-slide
cargo install --path .

# Present a markdown file
terminal-slide presentation.md

# Present HTML slides (opens browser)
terminal-slide slides.html --port 9000

Happy Presenting!
