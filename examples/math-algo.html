Algorithm: Binary Search

Pseudocode

function binarySearch(arr, target):
    lo ← 0
    hi ← len(arr) - 1

    while lo ≤ hi:
        mid ← ⌊(lo + hi) / 2⌋

        if arr[mid] == target:
            return mid
        else if arr[mid] < target:
            lo ← mid + 1
        else:
            hi ← mid - 1

    return -1  // not found

Algorithm: Quicksort

Pseudocode

function quicksort(arr, lo, hi):
    if lo < hi:
        p ← partition(arr, lo, hi)
        quicksort(arr, lo, p - 1)
        quicksort(arr, p + 1, hi)

function partition(arr, lo, hi):
    pivot ← arr[hi]
    i ← lo - 1
    for j ← lo to hi - 1:
        if arr[j] ≤ pivot:
            i ← i + 1
            swap(arr[i], arr[j])
    swap(arr[i+1], arr[hi])
    return i + 1
