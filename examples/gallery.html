Free Layout — Grid

メインビジュアル

大きなコンテンツエリア。画像、コード、図表などを配置。grid-row: 1/3 で2行分の高さを確保。

.grid-free {
  display: grid;
  grid-template-columns: 2fr 1fr;
  grid-template-rows: auto auto;
}
.span-2-rows {
  grid-row: 1 / 3;
}
